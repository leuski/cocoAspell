7.9 Creating An Official Dictionary Package

Once you have a basic dictionary working, you should consider creating an official package so that it can be distributed with Aspell. To do so download the aspell-lang package available at ftp://ftp.gnu.org/gnu/aspell/aspell-lang-version.tar.bz2 or in the “aspell-lang” module in the Aspell CVS repository available at https://savannah.gnu.org/cvs/?group=aspell. See the included README file for what to do. Or, send mail to aspell-dict at gnu org asking for help on how to get started.
